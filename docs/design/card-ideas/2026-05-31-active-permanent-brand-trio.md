## 0. 오늘 제안 개요

- **테마**: 낙인 (Brand) — 기존 임시 액티브 카드(Fear·Berserk 계열)의 픽률 저조(각각 13%·7%, QA 6차 리포트) 원인이 "효과가 10초 안에 사라진다"는 것. 낙인 트리오는 임시 효과와 함께 **영구 마크(낙인)** 를 동시에 남겨, 후반 픽에서도 의미 있는 이득을 제공한다.
- **목록**: 공포의 낙인 / 광기의 낙인 / 혈계 낙인
- **기존 25장 + git log 과거 회차와의 중복 회피 확인됨**
  - 기존 Fear = 3초 도망(임시 only). 기존 Berserk = 10초 +50% 공속(임시 only). 기존 Frenzy = 와일드(임시 only).
  - 낙인 카드들은 임시 효과 위에 **영구 스택/수치** 를 추가하는 이중 구조 — 기존 어느 카드도 이 패턴 없음.
  - 과거 1회차 (전장 상태 감지): 전장 상태를 읽어 스케일링 — 낙인 카드는 픽 자체가 영구 흔적을 쌓는 방식으로 트리거 축이 다름.
  - 과거 2회차 (종 간 연계 시너지): 종 공존 조건 보너스 — 낙인 카드는 픽 타이밍 의사결정 구조로 다름.
  - 과거 3회차 (플레이그-독 생태계): 플레이그 사망 이벤트 기반 독 체인 — 낙인 카드와 무관.

## 1. 공포의 낙인 (Brand of Fear) — 가칭

- **카테고리**: 액티브 저주
- **효과 모델**:
  - 영웅이 3초간 강제로 도망친다 (기존 Fear 와 동일한 임시 효과).
  - **동시에** 영웅에게 '공포 낙인' 1중첩을 영구 부여한다. 낙인 1중첩당 영웅 이동속도 **-5%** (영구·누적·최대 5중첩 → 최대 -25%).
  - 중첩 픽 시 낙인이 1씩 추가로 쌓인다. (같은 카드를 2번 픽 → -10%)
  - 밸런스 근거 (컨셉 §8): 이동속도 -25%는 기존 Slow(저주, -50% 10초)의 영구판 약화 버전. 영웅 AI 는 "가장 가까운 몬스터에게 이동"이므로 속도 저하 = 몬스터 공격 노출 시간 증가 = 수신 DPS 상승. 평균 사망 구간(120~240s) 내에서 -25% 이동속도는 누적 피해 약 +8~12% 증가로 추산.
- **구현 패턴**: `BrandOfFearAura : IHeroAura` 신규 클래스 (기존 HeroAuraRunner 파이프라인 활용). `stackCount` 필드 영구 보존, `IBattleContext.Hero.MoveSpeedMultiplier` 에 `1f - (stackCount * 0.05f)` 곱연산. 도망 트리거는 기존 FearEffect.cs 의 HeroFearService 그대로 재사용.
- **시너지 후크**:
  - Slow (기존 저주): 이미 -50% 이동속도 상태에서 낙인 -25%가 더해져 영웅이 거의 제자리에 묶임 → 집중 공격 조건 극대화.
  - BloodThirst (기존 버프): 느린 영웅 = 몬스터 처치 용이 → 처치 시 인근 몬스터 HP 회복 발동 빈도 증가.
- **구현 비용 추정**: 2 (HeroAura 스택 패턴 + FearService 재사용. 신규 시스템 없음)
- **중복 재검증**: 기존 Fear = 도망만, 영구 효과 없음. 낙인 개념(픽마다 누적되는 영구 수치)은 기존 25장 + 과거 3회차 어디에도 없음.

## 2. 광기의 낙인 (Brand of Madness) — 가칭

- **카테고리**: 액티브 버프
- **효과 모델**:
  - 10초간 현재 필드의 모든 몬스터 공격속도 **+30%** (임시, 기존 Berserk +50% 대비 -20%p 약화).
  - **버프 종료 시점** 에 필드에 살아있는 몬스터 종(種) 각각에 해당 종 공격속도 **+4% 영구 부여**.
  - 살아있는 종 수가 많을수록 영구 이득 증가 (최대 6종 × +4% = +24% 공속 영구).
  - 중첩 픽 시 임시 버프는 리셋(다시 10초), 영구 부여는 추가 누적 (종당 +4% 추가).
  - 밸런스 근거 (컨셉 §8): 리퍼 기본 공속 1회/1초. 낙인 2회 픽 + 6종 상태 = 리퍼 +8% 영구 → 0.926s/회. 미세하나, SpawnReapers·ReaperAtkSpeed 와 합산 시 공속 누적이 의미 있어짐. 광기의 낙인 단독으론 약하지만 다른 공속 버프와 시너지.
- **구현 패턴**: 기존 BerserkEffect.cs 에서 임시 버프 코드 재사용. 버프 종료 콜백에서 `IBattleContext.GetAliveMonsterSpecies()` 로 종 목록 조회 후 `MonsterBuffService.ApplyPermanentAtkSpeedBuff(species, 0.04f)` 호출. 종별 영구 버프는 기존 강화 카드(ReaperAtkSpeedEffect.cs) 패턴과 동일.
- **시너지 후크**:
  - SpawnX 카드들 (추가 계열 5장): 다양한 종의 Spawner 를 여러 개 가동 중이면 버프 종료 시 더 많은 종이 필드에 존재 → 영구 공속 획득 극대화.
  - ReplaceReapersToHex (기존 교체): 교체 후 헥스 Spawner 만 남으면 종 수 감소 → 낙인 카드 이득 감소. 전략 역상관 관계 생성.
  - ReaperAtkSpeed (기존 강화): 리퍼에게 이미 +30% 공속이 있는 상태에서 낙인으로 추가 +4%씩 쌓임 → 딜러 빌드 심화 경로.
- **구현 비용 추정**: 2 (BerserkEffect 재사용 + MonsterBuffService 영구 버프 메서드, GetAliveMonsterSpecies() 쿼리 추가 필요)
- **중복 재검증**: 기존 Berserk = 임시 +50%만. 과거 2회차 (종 간 연계 시너지) = 여러 종이 공존할 때 보너스 지급(버프 발동 조건). 낙인은 픽 타이밍 종료 후 스냅샷 보상 — 공존 조건 상시 확인 방식과 다름.

## 3. 혈계 낙인 (Brand of Blood) — 가칭

- **카테고리**: 액티브 와일드
- **효과 모델**:
  - 즉시 영웅 **현재 HP의 5%** 를 피해로 준다 (직접 HP 감소, 최소 1).
  - 동시에 현재 필드의 **모든 몬스터 최대 HP +10%** 영구 상향한다 (현재 필드 + 이후 스폰 동일하게 적용).
  - 최대 **3번** 중첩 픽 가능 (3번 픽 시: 영웅 총 15% 즉시 피해 + 몬스터 최대 HP 총 +30% 영구). 3번 초과 시 카드 풀에서 제거.
  - 밸런스 근거 (컨셉 §8): 영웅 HP 1000 기준 1회 픽 = -50 HP (즉각 손실) + 몬스터 탱킹 능력 +10%. 3회 픽 = -150 HP + 몬스터 +30% HP. 위스프 HP 200 → 260, 레이스 HP 500 → 650. 영웅 DPS 50으로 위스프 처치 시간 200/50=4s → 260/50=5.2s. 전체 몬스터 생존력 +30%는 사망 시간 약 +15~18% 기여 (비선형 감쇠 반영, QA §3.7 파워법칙 α=0.407 유추).
  - 리스크: 영웅 HP 200 이하에서 픽 시 -10 HP로 사망 유도 가능 → 타이밍 의사결정 필수.
- **구현 패턴**: `IBattleContext.Hero.TakeDamage(hero.CurrentHp * 0.05f)` (직접 피해, HeroPoisonAuraEffect.cs 의 DPS 적용 방식과 동일하나 1회 즉발). `MonsterBuffService.ApplyGlobalMaxHpBuff(1.10f)` (기존 WispHpBoostEffect.cs 의 타입별 버프를 전체 종으로 확장). 중첩 횟수 제한은 `CardEffectBase.maxStack = 3` 설정.
- **시너지 후크**:
  - WispHpBoost / WraithDamageBoost (기존 강화): 탱커 HP 이미 높은 상태에서 혈계 낙인 +10% = 절대값 이득 극대화 (위스프 HP 200+50%=300 → +10% = 330).
  - IronWill (기존 버프): 더 튼튼해진 몬스터 + 받는 피해 -30% 동시 적용 → 탱킹 시너지.
  - 광기의 낙인 (카드 2): 몬스터 HP 가 높아 오래 살아남으면 광기 낙인 버프 종료 시 살아있는 종 수 증가 → 영구 공속 이득 극대화.
- **구현 비용 추정**: 2 (기존 WispHpBoost(글로벌 버프 확장) + HeroPoisonAura(즉발 피해) 패턴 조합. 중첩 한도 카드 제거 로직이 신규이나 CardManager 범위 내 1~2줄 추가)
- **중복 재검증**: 기존 Frenzy = HP -50% + 데미지 +200% (임시 15초). 혈계 낙인 = 영웅 현재 HP 5% 즉발 피해 + 전체 몬스터 최대 HP 영구 +10%. 영구/임시 차원과 영웅 vs 몬스터 적용 대상 모두 다름.

## 4. 공통 테마 고찰

**왜 오늘 "낙인" 테마인가:**

QA 6차 리포트 §3.3 카드 픽률을 보면 Fear 13%, Berserk 7% — 하위 2자리. 기존 액티브 버프/저주의 구조적 약점은 "10초 지나면 아무 흔적 없음"이다. 플레이어 입장에서 "이번 30초 슬롯에 Fear vs Slow vs 아무 거나?"를 고민할 때, 일시적 효과만 있으면 기댓값 계산이 모호해진다.

낙인 트리오는 "픽할 때마다 영구 수치가 쌓인다"는 명확한 기댓값을 제공한다:
- 공포의 낙인: 픽 횟수 × (-5% 이동속도 영구)
- 광기의 낙인: 픽 종료 시점의 종 다양성 × (+4% 공속 영구)
- 혈계 낙인: 픽 횟수 × (+10% 몬스터 HP 영구) — 단, 최대 3번 제한

이 구조는 또한 **전략 분산 확장 (QA §3.4 문제)**에 기여한다. 낙인을 일찍 여러 번 픽하는 빌드 vs 후반에만 1번 픽하는 빌드 간 누적 수치 격차가 생긴다. span/mean이 현재 1.9%에서 개선될 가능성이 있다.

## 6. 쉬운 설명 (비개발자 요약)

지금 게임에서 활성화 카드(30초마다 뜨는 카드)들은 짧게 터지고 끝납니다. 예를 들어 "영웅이 3초 도망쳐라" 카드를 골라도 3초 지나면 아무 흔적이 없어요. 그러니 플레이어 입장에서 "이걸 골라봤자..." 싶어서 잘 안 고르게 됩니다.

오늘 제안하는 낙인 카드들은 짧은 효과가 끝난 뒤에도 **영구적인 흔적**을 남깁니다. 영웅에게 공포 도장을 찍어서 평생 조금씩 느리게 만들거나, 몬스터들이 일정 시간 빠르게 공격한 뒤 영구적으로 더 빠르게 되거나, 내 몬스터들의 체력을 영구적으로 올리는 대신 영웅이 지금 당장 피를 흘리는 카드입니다. 낙인은 여러 번 고를수록 쌓이기 때문에, 플레이어가 "이걸 몇 번 고를까?"를 고민하게 됩니다. 그래서 오늘 제안하는 카드 3장은: **"무언가 잃는 대신 영구적으로 쌓이는 이득"을 선택하는 의사결정 경험** 을 만드는 카드들입니다.
